# Tableau de bord KPI

## Objectif 2026-2027

Piloter la croissance du parcours professionnel sans affaiblir les demandes particuliers.

## KPI publics du site

| KPI | Source | Décision associée |
| --- | --- | --- |
| Clics `cta_quote_click` | GA4 existant via `data-event` | Mesurer l'attractivité du devis |
| Clics `phone_click` | GA4 existant via `data-event` | Mesurer l'intention urgente mobile |
| Clics `market_path_click` | Accueil | Comparer intérêt B2B et particuliers |
| Clics `machine_view` | Cartes machines | Identifier les formats les plus consultés |
| Clics `machine_compare` | Comparateur | Vérifier l'usage du comparateur |
| Clics `gallery_open` | Galerie | Mesurer le besoin de preuve visuelle |

## KPI internes à ne pas exposer

- Chiffre d'affaires professionnel.
- Objectif de chiffre d'affaires B2B.
- Taux de signature.
- Nombre de dossiers ouverts ou perdus.
- Captures ou données commerciales confidentielles.

## Lecture mensuelle

- Part de sessions entrant par `/evenements/corporates/`.
- Taux de clic devis sur les pages B2B.
- Taux de clic devis sur les pages particuliers.
- Machines les plus consultées avant devis.
- Requêtes SEO qui déclenchent impressions et clics.
